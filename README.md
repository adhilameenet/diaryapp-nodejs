# **Diary App**

## About 
A simple web app to write your daily, and save. This project is being done as part of Tinkerhub Co-Coder Program
<br><br>

## Status
Work in Progress...

## How to Run


```
git clone https://github.com/adhilameenet/diaryapp-nodejs

```

```
cd diaryapp-nodejs

```
```
npm install

```
### Setup .env file
